# URBANIX

An urban planning and liveability analysis platform with AI-powered decision support.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- Ollama (for local AI model support)

### Installation

1. Install dependencies:
```bash
npm i
```

2. Set up Ollama:
```bash
curl https://ollama.ai/install.sh | sh
ollama pull tinyllama
```

3. Start the Ollama server:
```bash
ollama serve
```

4. In another terminal, start the backend server:
```bash
npm run server
```

5. In another terminal, start the development server:
```bash
npm run dev
```

## Features

- Liveability Dashboard
- AI Decision Layer
- Policy Simulation
- Risk Mapping
- Data Import
- Interactive Chatbot
